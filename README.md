# Expense Tracker System

A modern, full-stack expense tracking application designed to help individuals take control of their finances through smart budgeting, insightful analytics and intelligent automation.

**Live Demo:** https://expense.abongo.co.ke  
*(Create an account, log in and explore the system)*

---

## Overview

The Expense Tracker System enables users to record, categorize and monitor their expenses in real time. Beyond basic tracking, the system provides budgeting tools, alerts, AI-driven spending predictions, downloadable reports and personalized insights to help users make better financial decisions.

This project focuses on **usability**, **performance**, and **scalability**, while keeping user data secure and private.

---

## Key Features

### Expense Management
- Record and categorize daily expenses
- Tie expenses directly to budgets
- View detailed expense history and breakdowns

### Budgeting & Alerts
- Create and manage budgets per category
- Automatic alerts when spending exceeds limits
- In-system notifications and email alerts
- Respond to alerts by:
  - Increasing budget limits
  - Deactivating alerts
  - Reviewing spending trends

### Reports & Insights
- Download expense reports in **Excel format**
- Personalized spending summaries
- Monthly and category-based analytics

### AI-Powered Intelligence
- AI predictions based on spending habits
- Future expense trend analysis
- Smart insights to guide better financial planning

### Security & Authentication
- Secure authentication system
- User data isolation and privacy
- Protected API endpoints

### Notifications
- In-app notifications
- Email alerts for key financial events

## Getting Started (Local Development)

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn
- A configured database (as per backend setup)

### Installation
```bash
# Clone the repository
git clone https://github.com/Mark-crypto/Expense_Tracker.git

# Navigate to project directory
cd expense-tracker

# Install dependencies (frontend & backend)
npm install
```

### Run the Application

```bash
# Start backend
npm run dev

# Start frontend
npm run dev
```
